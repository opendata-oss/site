# OpenData Site

Static homepage for [OpenData](https://github.com/opendata-oss/opendata) - open-source databases for object storage.

## Tech Stack

- Astro.js
- TailwindCSS v4

## Commands

```bash
npm run dev      # Start dev server
npm run build    # Build to dist/
npm run preview  # Preview build
```

## Structure

```
src/
  pages/index.astro       # Homepage
  components/icons/       # Icon components (GitHub.astro, Discord.astro)
  styles/global.css       # Tailwind imports
public/images/            # Logo assets (SVG)
```
